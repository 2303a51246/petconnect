# 🐾 PetConnect - Marketplace Web Application

PetConnect is a full-stack marketplace platform where users can buy and sell products with features like authentication, product listings, and user interaction.

---

##  Features

-  User Authentication (Login/Register)
-  Product Listings with Images
-  Wishlist Feature
-  Chat System (if implemented)
-  Notifications
-  Admin Panel
-  Image Uploads

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-time: Socket.io (if used)

## ⚙️ Installation

1. Clone the repository:
git clone https://github.com/2303a51246/petconnect.git  

2. Install dependencies:
npm install  

3. Create `.env` file:
PORT=5000  
MONGO_URI=your_mongodb_connection  
JWT_SECRET=your_secret  

4. Run the project:
npm start
